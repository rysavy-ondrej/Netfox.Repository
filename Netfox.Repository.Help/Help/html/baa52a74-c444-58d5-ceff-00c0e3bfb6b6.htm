Netfox.Repository Namespace

[This is preliminary documentation and is subject to change.]

The Netfox.Repository namespace contain classes for accessing and managing data using the approach similar to Entity Framework. Main class is [NRepositoryContext] that enables to create and manage data context in your application.
Classes
  ClassDescription
ContextStatistics
Provides information and statistics about the usage of Repository Context.
NCollectionEntryTDocument, TElement
Instances of this class are returned from the Collection method of NDocumentEntry and allow operations such as loading to be performed on the an document's collection navigation properties.
NDocumentEntry
An abstract class that represents untyped NDocumentEntry object. This object is used by users to get metainformation about the associated document.
NDocumentEntryTDocument
Instances of this class provide access to information about and control of documents that are being tracked by the NRepositoryContext. Use the Document or Documents methods of the context to obtain objects of this type.
NDocumentSetTDocument
An NDocumentSet represents the collection of all entities in the context, or that can be queried from the database, of a given type. NDocumentSet objects are created from a NRepositoryContext using the NRepositoryContext.Collection method.
NHashSetTDocument
Represents a set of documents of the specified type.
NObservableCollectionTDocument
An observable collection is implementation of a push-based enumeration of documents.
NPropertyEntryTDocument, TProperty
Instances of this class are returned from the Property method of NDocumentEntry.
NReferenceEntryTDocument, TProperty
Instances of this class are returned from the Reference method of NDocumentEntry and allow operations such as loading to be performed on the an document's reference navigation properties.
NRepositoryContext
This class provides access to underlying data repository represented by Mongo database and maintains document states.
