Examples
Usage of NReferenceAttribute is shown in the following example:
Copy
[NDocument]
public class Message  
{ 
    [NReference]
    public User Sender { get; set; }

    [NCollection(typeof(User)] 
    public ICollection{User} Receivers { get; set; }

    [NValue]
    public string Text { get; set; }
}
Here, Sender is a users that sent the message. It is a reference to the User object. It is persisted as oid. The Netfox.Repository performs this automatically when the property is annotated with NReferenceAttribute.
