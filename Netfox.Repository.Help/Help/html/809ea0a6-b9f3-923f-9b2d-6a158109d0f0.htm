Netfox.Repository.Serialization Namespace

[This is preliminary documentation and is subject to change.]

The Netfox.Repository.Serialization namespace contains crafted Bson serializers that supports Netfox.Repository data management framework.
Classes
  ClassDescription
NDocumentSerializer
An abstract base class of serializers for all classes that implements IDocument interface.
NDocumentSerializerTDocument
Implements a custom serializer for persistent documents.
Delegates
  DelegateDescription
AfterDocumentDeserializationCallback
This callback is used to inform an observer that a new document has been desrialized. It can be used to add this object to the cache.
BeforeDocumentDeserializationCallback
This callback is used to inform an observer that a new document is to be deserialized. Using this callback it is possible to skip deserialization by providing cached object as the result of this callback.
