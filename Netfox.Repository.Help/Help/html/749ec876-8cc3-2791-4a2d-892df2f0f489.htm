Netfox.Repository.Attributes Namespace

[This is preliminary documentation and is subject to change.]

The Netfox.Repository.Attributes namespace contains attribute definitions that data object should use to annotate contolled properties and refefences.
Classes
  ClassDescription
NCollectionAttribute
This attribute should be used on data object to annotate properties that are collection of references.
NDocumentAttribute
This attribute should be used with POCO objects serialized to MongoDB. Using this attribute persistent aspect is applied to the target classs.
NReferenceAttribute
This attribute should be used on data object to annotate properties that are references.
NValueAttribute
Marks the annotated property as document value. Such property will have a field in the database.
