Namespaces
NamespaceDescription
Netfox.Repository
The Netfox.Repository namespace contain classes for accessing and managing data using the approach similar to Entity Framework. Main class is [NRepositoryContext] that enables to create and manage data context in your application.
Netfox.Repository.Attributes
The Netfox.Repository.Attributes namespace contains attribute definitions that data object should use to annotate contolled properties and refefences.
Netfox.Repository.Serialization
The Netfox.Repository.Serialization namespace contains crafted Bson serializers that supports Netfox.Repository data management framework.
Netfox.Repository.Utils
The Netfox.Repository.Utils namespace contains various classes that are used throrough the Netfox.Repository library and can be used also in user applications.
