Remarks
Except the access to the MongoDB data store, this class performs document state management. An application usually contains a single instace of the NRepositoryContext to consistently manage the data documents. NRepositoryContext is created by specifying a connection string to the databse server and the name of the database at that server. Each context can be associated only with a sibngle database. Documents in No-SQL data model live in collections. Use this class to access collections in the database associated with the current NRepositoryContext. A collection is represented by NDocumentSetTDocument class that provides method for accessing individual documents.
Examples
NRepositoryContext can be used to write data to the data store as follows:
Copy
using (var ctx = NRepositoryContext.Create("DataBase"))
{
    var usersCollection = ctx.Collection<Users>();
    var newUser = new User {Name = "John Doe"};
    usersCollection.Add(newUser);
    await ctx.SaveChangesAsync();
}
Examples
NRepositoryContext can be used to read data from the data store as follows:
Copy
using (var ctx = NRepositoryContext.Create("DataBase"))
{
    var usersCollection = ctx.Collection<Users>();

    var john = await fooCollection.FindOneAsync(user => user.Name.Contains("John"));

    if (john != null)
    {
        // do something with a retrieved document
    }
}
